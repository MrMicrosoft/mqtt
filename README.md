# MQTT Chat
## REST Backend
Implemented with NodeJS in the MVC Model. To connect it with an mongodb Database i used mongoose.

- cors 2.8.4
- express 4.16.3
- moment 2.22.2
- mongoose 5.1.3
- mongoose-auto-increment 5.0.1
